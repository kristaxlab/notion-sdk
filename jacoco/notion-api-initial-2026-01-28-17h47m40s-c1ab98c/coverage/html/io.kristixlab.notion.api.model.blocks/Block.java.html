/**
 * Abstract base class representing a Notion block.
 *
 * <p>Blocks are the fundamental building blocks of content in Notion. They can contain text, media,
 * or other structured content. This class serves as the base for all specific block types and
 * provides common properties and functionality.

@EqualsAndHashCode(callSuper = true)

  @JsonProperty("type")
